# Wohl Backend API

Starter Express API for migration from PHP backend logic.

## Endpoints

- `GET /api/v1/health`
- `GET /api/v1/products`
- `GET /api/v1/cart`
- `POST /api/v1/cart/items`
- `DELETE /api/v1/cart/items/:id`
- `GET /api/v1/wishlist`
- `POST /api/v1/wishlist/items`
- `POST /api/v1/auth/login`
- `POST /api/v1/auth/register`
- `POST /api/v1/orders/checkout`

## Commands

```bash
npm install
npm run dev
```
